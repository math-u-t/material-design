## ğŸš€ Quick Start

### CDN Usage (via jsDelivr)

Add one of the following lines to your HTML `<head>`:

```html
<!-- Blue Sea Theme -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/math-u-t/material-design@main/style/blue-sea.css">

<!-- Or any other theme -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/math-u-t/material-design@main/style/[THEME_NAME].css">
```

### GitHub Raw Content

```html
<link rel="stylesheet" href="https://raw.githubusercontent.com/math-u-t/material-design/main/style/[THEME_NAME].css">
```

### Download

Download the CSS file directly from the repository and include it in your project:

```html
<link rel="stylesheet" href="path/to/[THEME_NAME].css">
```

## ğŸ¨ Customization

All themes use CSS Variables for easy customization:

```css
:root {
  --color-primary: #YourColor;
  --font-primary: 'Your Font', sans-serif;
  --spacing-md: 1rem;
  --radius-md: 0.5rem;
}
```

Simply override the variables in your own stylesheet to customize the theme.

## ğŸ¨ Choosing a Theme

Consider these factors when selecting a theme:

1. **Industry**: Match your theme to your industry (e.g., Green Supa for eco-friendly, Navy Professional for corporate)
2. **Brand Colors**: Choose a theme that complements your existing brand
3. **Mood**: Consider the emotional response you want (e.g., Lavender Calm for wellness, Fire Red for energy)
4. **Accessibility**: All themes meet WCAG AA standards, but lighter themes may be easier for extended reading
